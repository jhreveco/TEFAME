- Método de rigidez, análisis estático
- Método modal espectral, análisis estático

## Tipo de elementos

- Biela 2D
- Biela 3D
- Membrana (Shell 2D)
- Viga 2D (c/s masa)
- Viga-columna 2D

- Viscoso 2D
- Triangular 2D
- Friccional puro 2D
